How to run on local env:

1. clone the repo
2. if using new DB : cd backend -> flask db init -> flask db migrate -m "Initial migration" -> flask db upgrade
3. cd backend -> pip install -r requirements.txt ->python run.py
4. cd frontend -> npm install -> npm run dev
